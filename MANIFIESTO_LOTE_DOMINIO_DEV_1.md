# Manifiesto del lote dominio DEV 1

## Objeto
Este lote declara el dominio especializado `𝔇_DEV` y añade su evaluador algebraico inicial al frente motor del Sistema Vectorial SV.

## Qué incorpora
- `src/sv_motor/algebra/dev.py`;
- pruebas del dominio DEV;
- laboratorio mínimo del dominio DEV;
- declaración formal y documento de arquitectura del dominio;
- actualización de API pública, deuda viva y hoja de cambios.

## Qué no incorpora
- capa generativa DEV;
- búsqueda web operativa del dominio;
- extractor DEV;
- activación de modelos externos.

## Criterio de fase
Primero evaluador, después generador. Cualquier apertura futura del dominio DEV queda subordinada a esta capa evaluadora.
