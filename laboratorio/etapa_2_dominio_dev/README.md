# Laboratorio mínimo del dominio DEV — Etapa 1

Este laboratorio demuestra la existencia material del dominio `𝔇_DEV` como dominio evaluable del motor.

## Qué acredita
- transductor `I_DEV` sobre una célula `C_dev^9`;
- evaluación algebraica determinista del dominio;
- preservación de `U` sin clausura falsa;
- y apertura deliberadamente no activada de la capa generativa del dominio.

## Ejecución
```bash
python laboratorio/etapa_2_dominio_dev/ejecutar_laboratorio_dev_etapa_1.py
```
