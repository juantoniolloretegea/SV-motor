Esta carpeta reúne los bloques reproducibles del repositorio `SV-motor`.

## Etapas actuales

- `etapa_0_preparacion/` — documentación preparatoria del frente.
- `etapa_0_demostracion_local/` — demostración end-to-end local.
- `etapa_1_nucleo_local/` — laboratorio mínimo reproducible del núcleo local soberano.
- `etapa_2_dominio_dev/` — evaluador algebraico inicial del dominio `𝔇_DEV`.
- `etapa_3_custodia_estructural/` — compuerta ejecutable mínima de custodia del frente motor.
- `etapa_4_puente_svp/` — reservado para el puente formal a `.svp`.
- `etapa_5_verificacion_python/` — runner, comparador y doble vara local.

La existencia de esta carpeta no equivale a laboratorio completo del motor. Su función actual es acreditar una base real, trazable y revisable por terceros.
