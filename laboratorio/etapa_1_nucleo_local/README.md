Esta etapa acredita una base mínima reproducible del frente motor.

## Objetivo

Demostrar, con casos canónicos y salidas serializadas, que:

1. la evaluación algebraica local existe y es ejecutable;
2. la compuerta conserva la U cuando no hay base legítima de cierre;
3. la capa extractora directa valida el dominio y no fabrica certeza;
4. la salida del laboratorio es legible por terceros.
