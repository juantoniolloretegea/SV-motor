# Contrato de actores y extractores auxiliares del Sistema Vectorial SV

Toda capa auxiliar que produzca observables debe respetar cinco invariantes:

1. no redefinir `T(n)` ni `κ₃`;
2. no producir cierre soberano;
3. emitir `U` cuando falte base suficiente;
4. validar claves, dominios y codominios antes del motor;
5. dejar trazabilidad suficiente para auditoría de terceros.
