# Relación con el Lenguaje SV

Este repositorio no modifica la gramática, la IR, el validator, el lowering ni el backend del Lenguaje SV.

## Regla de trabajo

Cualquier necesidad que exija alterar:

- gramática superficial,
- IR,
- runner,
- validator,
- lowering,
- o backend soberano,

debe detener el avance local del motor y elevar la cuestión al carril correspondiente.
