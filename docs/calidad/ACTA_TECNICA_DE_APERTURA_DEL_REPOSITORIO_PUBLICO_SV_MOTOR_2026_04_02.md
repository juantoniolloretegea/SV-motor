# Acta técnica de apertura del repositorio público SV-motor — 2026-04-02

## Objeto

Dejar asentada la apertura pública del repositorio `SV-motor` como sede de estudio y materialización mínima del frente motor del Sistema Vectorial SV.

## Alcance

La apertura cubre estructura pública, documentación de gobierno, documentación de arquitectura, instrumentos mínimos de calidad, paquete Python local y laboratorio inicial de Etapa 1.

## Límites

La apertura no equivale a motor completo, no abre backend soberano, no cierra la integración con el Lenguaje SV y no declara madurez suficiente del frente NLP.

## Dictamen

Apertura procedente, con deuda viva explícita sobre laboratorio ampliado, sincronización fuerte con sedes superiores y extractores subordinados.
