# Checklist de custodia del motor

- [ ] El repositorio declara su dependencia doctrinal.
- [ ] El paquete local instala de forma reproducible.
- [ ] El criterio ternario está escrito y visible.
- [ ] No-aplica y U aparecen separados.
- [ ] No hay atajos opacos fuera del código, tablas y laboratorio.
- [ ] Los tests básicos están en verde.
- [ ] Existe laboratorio mínimo reproducible.
- [ ] Repositorio, documento y laboratorio no se contradicen.
- [ ] La pieza no promete cobertura total ni backend ya resuelto.
- [ ] El cierre no depende de un servicio externo.
