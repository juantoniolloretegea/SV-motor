# Checklist de custodia del motor

- [x] El repositorio declara su dependencia doctrinal.
- [x] El paquete local instala de forma reproducible.
- [x] El criterio ternario está escrito y visible.
- [x] No-aplica y U aparecen separados.
- [x] No hay atajos opacos fuera del código, tablas y laboratorio.
- [x] Los tests básicos están en verde.
- [x] Existe laboratorio mínimo reproducible.
- [x] Repositorio, documento y laboratorio no se contradicen.
- [x] La pieza no promete cobertura total ni backend ya resuelto.
- [x] El cierre no depende de un servicio externo.
