# Arranque de agentes especializados y realimentación experta

## Objeto

Describir la fase de arranque de un agente especializado y el bucle posterior de aprendizaje auxiliar.

## Fase de arranque

Un agente especializado no nace desde vacío. Nace a partir de:

1. fuentes materiales del dominio;
2. decisión experta sobre parámetros pertinentes;
3. criterio ternario explícito por posición;
4. capas, puentes y meta-células cuando proceda;
5. y construcción de casos iniciales suficientes para operar.

La fase de arranque puede apoyarse en la Inteligencia Lógica, pero su cierre sigue siendo humano y soberano.

## Bucle posterior

Una vez arrancado el dominio, el sistema trabaja así:

1. entra un nuevo caso, frame o imagen;
2. la Inteligencia Lógica y, cuando proceda, la capa visual especializada formulan sugerencias;
3. el experto revisa;
4. corrige una o varias `U` si procede;
5. nace un nuevo suceso;
6. y las capas auxiliares aprenden de la corrección soberana.

## Regla de aprendizaje

El aprendizaje permitido es **post-corrección** y **orientado a consejo futuro**.

No se permite aprendizaje para:
- fundar verdad;
- cerrar `U` por sí solo;
- ni desplazar el cierre experto.
